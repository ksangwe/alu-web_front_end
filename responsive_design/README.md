# Responsive Design

Front-end web development project building a responsive version of the
Techium homepage using mobile-first design, media queries, and flexible
CSS units.

## Files

* `00-index.html` / `00-styles.css` - Starter homepage
* `01-index.html` / `01-styles.css` - Hero banner fix (background position, size, hero height)

## Images

Basic assets (banner, logos, favicon) live in the `images/` directory.
